# Code of Conduct

## Our Pledge

We want Aevor to be a welcoming project for anyone who wants to use it, report a bug, or contribute code. Everyone interacting in this project's issues, pull requests, and discussions is expected to treat others with respect.

## Expected Behavior

- Be respectful and considerate in how you communicate.
- Give constructive feedback, and be open to receiving it.
- Focus disagreements on ideas and implementation, not people.
- Assume good intent, and ask for clarification before assuming the worst.

## Unacceptable Behavior

- Personal attacks, insults, or derogatory comments
- Harassment of any kind, public or private
- Publishing others' private information without consent
- Trolling, deliberately derailing discussions, or spamming issues and pull requests

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any other communication channels tied to the project.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please open an issue or contact a maintainer directly. Reports will be reviewed and handled as appropriate, with confidentiality respected wherever possible.

## Enforcement

Maintainers are responsible for clarifying and enforcing these standards. This may include editing or removing comments, commits, or contributions that don't align with this Code of Conduct, or temporarily or permanently banning contributors for behavior deemed inappropriate or harmful.
