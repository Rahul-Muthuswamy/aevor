# Aevor

Aevor is an open-source Windows desktop app for managing Brave Browser profiles. It lets you turn any profile into a reusable template so you can clone it, share it, or move it to another machine without starting from scratch every time.

It's currently in early development, focused on understanding how Brave profiles are structured and figuring out what's safe to copy versus what should never leave your machine.

## What it does

Discover and analyze your Brave profiles, create reusable templates from them, clone configurations to new profiles, and export or import templates across devices.

## Security

Aevor moves your browser environment, not your identity. Passwords, cookies, session tokens, wallet data, and payment information are never exported, no exceptions.

## Status

Early research phase. Current focus is mapping Brave's profile structure, identifying portable vs sensitive files, and designing the template format.

Roadmap: Profile Discovery, Profile Analysis, Sensitive Data Scanner, Template Creation, Import/Export, Profile Cloning, Profile Comparison, Community Template Sharing.
